
  Dia-mina Chic Couture – Mini Site E-Commerce
  Projet Web – L2 Informatique – UMMTO

DESCRIPTION :
  Mini site e-commerce thématique "Robes de Soirée".
  5 produits par catégorie (Bal, Cocktail, Mariage, Gala).
  Authentification simulée avec localStorage.
  Panier fonctionnel, recherche dynamique, formulaires validés.

INSTRUCTIONS D'UTILISATION :
  1. Ouvrir index.html dans un navigateur (double-clic ou Live Server)
  2. Naviguer via le menu en haut
  3. Compte de test : dihia@ummto.dz / Dihia2025!
  4. Ou créer un nouveau compte via "Inscription"

STRUCTURE DU PROJET :
  DA_Couture/
  ├── index.html                  → Page d'accueil
  ├── readme.txt                  → Ce fichier
  ├── content/
  │   ├── produits.html           → Catalogue avec filtres
  │   ├── inscription.html        → Formulaire d'inscription
  │   ├── connexion.html          → Formulaire de connexion
  │   └── commande.html           → Panier & commande
  ├── style/
  │   ├── global.css              → Variables & base
  │   ├── navbar.css              → Navigation & barre de recherche
  │   ├── hero.css                → Section hero & catégories
  │   ├── products.css            → Cartes produits & filtres
  │   ├── forms.css               → Formulaires (inscription/connexion)
  │   ├── commande.css            → Page panier
  │   └── footer.css              → Pied de page & contact
  ├── javascript/
  │   ├── data.js                 → Données produits (tableau JS)
  │   ├── search.js               → Barre de recherche dynamique
  │   ├── main.js                 → Logique page d'accueil
  │   ├── produits.js             → Catalogue + filtres par catégorie
  │   ├── auth.js                 → Authentification + RegEx validation
  │   └── commande.js             → Gestion du panier
  └── images/                     → (dossier pour vos images)
